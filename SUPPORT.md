# Support

Need help with Ainfera? Here's how to reach us.

## Channels

| Channel | Use it for |
|---------|-----------|
| **GitHub Issues** | Bug reports, feature requests, reproducible problems in a specific repo |
| **GitHub Discussions** | Questions, ideas, show-and-tell, help from the community |
| **Email** — [hello@ainfera.ai](mailto:hello@ainfera.ai) | Partnership, private questions, anything that doesn't fit a public issue |

## Before opening an issue

1. Check the repo's README and docs at [ainfera.ai](https://ainfera.ai).
2. Search [existing issues](https://github.com/ainfera-ai) — your question may already be answered.
3. Open the issue in the repo closest to the problem (`cli`, `sdk-python`).

## Security

Do NOT report security vulnerabilities through public issues. See [SECURITY.md](SECURITY.md).

## Response times

- GitHub Issues / Discussions: best-effort, usually within a few business days.
- `hello@ainfera.ai`: within 2 business days.
- Security reports: within 24 hours (see SECURITY.md).
